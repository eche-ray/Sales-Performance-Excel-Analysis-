This project analyzes sales performance data using Microsoft Excel to uncover revenue trends, top-performing products, and regional insights. An interactive dashboard was created to support data-driven decision-making.

Business Objectives
- Identify top-performing products and regions
- Analyze revenue trends over time
- Provide an interactive dashboard for stakeholders
- Summarize key sales metrics using KPIs

Dataset
- 500 rows of sales transaction data
- Includes product, region, date, quantity, pricing, and revenue information

Data Cleaning
- Standardized date formats
- Removed missing values
- Recalculated revenue
- Created Year and Month columns
- Converted data into an Excel Table

 Analysis & Dashboard
The Excel dashboard includes:
- Total Revenue, Total Quantity, and Average Order Value KPIs
- Monthly Revenue Trend (Line Chart)
- Revenue by Product (Bar Chart)
- Revenue by Region (Bar Chart)

Tools Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Excel Tables
